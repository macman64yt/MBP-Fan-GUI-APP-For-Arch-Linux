# MBP-Fan-GUI-APP-For-Arch-Linux
This is a app that simplifys mbpfan into a simple gui app
mbpfan Fan Control App This folder contains a simple Python GUI wrapper for the mbpfan terminal program. Requirements mbpfan installed on your system python3 tkinter support (tk package on Arch) pystray pillow Install the tray dependencies with: pip install -r mbpfan/requirements.txt

Arch Linux On Arch, install the Tk toolkit and then install mbpfan from the AUR or your preferred AUR helper: sudo pacman -S tk git base-devel git clone https://aur.archlinux.org/mbpfan.git cd mbpfan makepkg -si  If you use an AUR helper like yay, you can run: yay -S mbpfan

Usage Install mbpfan on your Linux system. Run the GUI app: python3 mbpfan/fan_control.py  3. Use the buttons to start or stop mbpfan, refresh status, adjust CPU/HDD/GPU fan RPM sliders, and inspect the config file. If you want the app to appear in Rofi, install the desktop launcher: mkdir -p ~/.local/share/applications cp mbpfan/mbpfan-fan-control.desktop ~/.local/share/applications/ chmod +x mbpfan/run_fan_control.sh  Then reopen Rofi; the app should show up as “mbpfan Fan Control”. The slider ranges are tuned for a 2011 21.5" iMac: CPU fan: 1200–6200 RPM HDD fan: 1200–5200 RPM GPU fan (disk drive): 1200–6200 RPM These sliders now write real mbpfan settings into /etc/mbpfan.conf using min_fan*_speed and max_fan*_speed values. Notes On many systems, mbpfan requires root privileges to start and stop. The app uses systemctl when available, otherwise it falls back to direct mbpfan invocation. The default config path is /etc/mbpfan.conf.
